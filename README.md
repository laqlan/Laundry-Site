# LaundromatGreece Website

A modern, responsive website for LaundromatGreece - Lefkada's premier laundry and cleaning service provider.

## Features

- **Modern Design**: Fresh, clean design with gradient accents using the brand's blue color scheme
- **Fully Responsive**: Optimized for all devices from mobile to desktop
- **Multiple Pages**: 
  - Homepage with services overview and testimonials
  - Dedicated Laundry Services page
  - Property Cleaning & Housekeeping page
  - Contact page with form and location information
- **Cohesive Typography**: Inter font family throughout for a professional, modern look
- **Gradient Highlights**: Blue gradient text effects on headers and CTAs
- **All Original Images**: All images from the existing website are integrated
- **24/7 Availability**: Prominently displayed throughout the site
- **Two Locations**: Clear information about both Lefkada Town and Nydri locations

## Design Elements

### Color Scheme
- Primary: Blue gradients (from-blue-600 to-blue-800)
- Accent: Cyan to blue gradients for highlights
- Background: Clean whites and light grays
- Text: Dark grays for readability

### Typography
- Font Family: Inter (Google Fonts)
- Weights: 300-900 for varied hierarchy
- Gradient text on headers for visual impact

### Components
- Sticky navigation with mobile menu
- Hero sections with gradient backgrounds
- Service cards with hover effects
- Testimonial cards with 5-star ratings
- Contact form with validation
- Interactive location cards
- Call-to-action buttons with gradients
